# 项目 005: AI 对联生成器 - 详细方案

## 📖 项目概述

**项目名称**: AI 对联生成器 (AI Couplet Generator)

**一句话介绍**: 输入上联/主题，AI 秒对下联，传承传统文化的智能对联工具

**推荐优先级**: Top 4

**核心价值**: 结合 AI 能力与传统文化，降低对联创作门槛，满足春节、婚庆、开业等场景需求，兼具教育意义和实用价值。

---

## 🎮 详细玩法说明

### 核心功能

#### 1. AI 对对联

**使用场景**:
- 用户输入上联 → AI 对下联 + 横批
- 用户输入主题 → AI 生成完整对联
- 用户输入关键词 → AI 创作藏头对联

**对仗规则**（AI 遵循）:
- 字数相等
- 词性相对（名对名、动对动）
- 平仄相协（传统要求，可放宽）
- 意义相关或相反
- 避免重复字

**风格选择**:
- **传统经典**: 严格遵循格律
- **现代通俗**: 通俗易懂
- **幽默风趣**: 网络梗、段子风格
- **行业定制**: 商业、婚庆、寿诞等

#### 2. 对联知识库

**内容模块**:
- **经典名联**: 历史名联赏析
- **对联知识**: 格律、技巧讲解
- **节日专题**: 春节、元宵、中秋等
- **场景分类**: 春联、婚联、寿联、挽联等

**学习功能**:
- 对联解析（词性、平仄标注）
- 创作技巧教学
- 每日一对（学习 + 练习）

#### 3. 互动玩法

**对联 PK**
- 系统出上联 → 用户对下联 → AI 评分
- 好友对战 → 互相出题 → 投票评选
- 排行榜 → 积分排名 → 称号系统

**对联接龙**
- 多人参与，每人接一句
- 最后一字作为下一句首字
- AI 辅助评判是否合规

**每日挑战**
- 每日 3 题，难度递增
- 完成得积分
- 连续打卡奖励

### 进阶玩法

#### 1. 定制化服务

**企业定制**
- 公司年会对联
- 开业庆典对联
- 产品宣传对联

**个人定制**
- 婚庆对联（新人姓名藏头）
- 寿诞对联（年龄、成就融入）
- 新居对联（地址、寓意融入）

#### 2. 社交分享

**分享形式**
- 对联卡片（红底金字）
- 书法风格图片（多种字体）
- 短视频（AI 朗读 + 动画）

**传播机制**
- "测测你的对联水平"
- "为 TA 创作一副对联"
- "家乡春联大 PK"

#### 3. 线下结合

**AR 写春联**
- 手机摄像头对准红纸
- AR 显示对联内容
- 可临摹书写

**打印服务**
- 在线定制对联
- 快递到家
- 配套福字、窗花

### 核心 API 设计

```typescript
// 1. AI 对对联
POST /api/v1/couplet/generate
Body: {
  mode: "complete" | "response" | "acrostic",
  style: "traditional" | "modern" | "humorous",
  input: "输入内容",
  context?: "背景信息"
}
Response: {
  couplets: Array<{
    upper: string,
    lower: string,
    horizontal: string,
    score: number,
    analysis: object
  }>
}

// 2. 对联评分
POST /api/v1/couplet/score
Body: { upper, lower, horizontal }
Response: {
  score: number,
  feedback: string,
  suggestions: Array
}

// 3. 生成书法图片
POST /api/v1/couplet/image
Body: { couplet, font, style, background }
Response: { imageUrl }

// 4. PK 挑战
POST /api/v1/challenge/submit
Body: { challengeId, userLower }
Response: {
  isCorrect: boolean,
  score: number,
  aiLower: string,
  analysis: string
}
```

### 数据库设计

```sql
-- 用户表
CREATE TABLE users (
  id VARCHAR(32) PRIMARY KEY,
  openid VARCHAR(64),
  nickname VARCHAR(50),
  points INT DEFAULT 0,
  level VARCHAR(20),
  created_at TIMESTAMP
);

-- 对联记录表
CREATE TABLE couplets (
  id VARCHAR(32) PRIMARY KEY,
  user_id VARCHAR(32),
  upper TEXT,
  lower TEXT,
  horizontal VARCHAR(50),
  style VARCHAR(20),
  source VARCHAR(20), -- ai/user
  score INT,
  created_at TIMESTAMP,
  INDEX idx_user (user_id)
);

-- 挑战表
CREATE TABLE challenges (
  id VARCHAR(32) PRIMARY KEY,
  upper TEXT,
  standard_lower TEXT,
  difficulty VARCHAR(20),
  category VARCHAR(50),
  attempt_count INT DEFAULT 0,
  created_at DATE
);

-- 挑战记录表
CREATE TABLE challenge_records (
  id VARCHAR(32) PRIMARY KEY,
  challenge_id VARCHAR(32),
  user_id VARCHAR(32),
  user_answer TEXT,
  score INT,
  time_used INT,
  created_at TIMESTAMP,
  INDEX idx_user (user_id)
);
```

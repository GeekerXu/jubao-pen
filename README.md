# 聚宝盆 (Jubao Pen)

> 一个汇聚 100 个深度调研项目的知识宝库

## 📖 项目介绍

聚宝盆是一个系统化的项目调研与知识管理平台。我们致力于对 100 个有价值的主题进行深度调研，形成结构化的知识体系，并通过 HTML 页面的形式呈现，让知识更易获取和传播。

"聚宝盆"寓意汇聚珍宝——在这里，每一个项目都是一颗珍珠，串联起来就是知识的宝库。

## ✨ 项目亮点

- **🎯 100 个项目计划**：覆盖技术、商业、文化、科学等多个领域
- **📚 深度调研**：每个项目都经过系统性研究，提供深度洞察
- **🌐 静态页面**：所有成果以 HTML 形式呈现，无需服务器即可浏览
- **📂 结构化组织**：清晰的目录结构，便于导航和扩展
- **🤝 开源协作**：欢迎社区贡献，共同完善知识体系

## 🚀 快速开始

### 浏览 HTML 页面

1. **本地浏览**：
   ```bash
   # 克隆仓库
   git clone https://github.com/your-username/jubao-pen.git
   
   # 直接在浏览器中打开 html/index.html
   # 或使用本地服务器
   cd jubao-pen
   python -m http.server 8000
   # 访问 http://localhost:8000/html/
   ```

2. **在线浏览**：
   - 访问 GitHub Pages 部署的网站（自动部署）
   - 链接：`https://your-username.github.io/jubao-pen/`

### 查看项目计划

所有项目计划位于 `plans/detailed/` 目录，每个 Markdown 文件包含一个项目的详细规划。

### 阅读调研资料

调研资料存放在 `research/` 目录，按主题分类组织。
